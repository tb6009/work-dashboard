Stage 6 — 5/3 오후 | design2 (현재)
하루 캐릭터 + 퍼포먼스 대시보드
모미+마음 → "하루" 통합 비서. 기록 탭에 Today/Week/Month 서브 탭. 블루 계열 통일.
  • 하단 4탭: 홈 / 하루(Assistant) / 기록(Diary) / 설정(Settings)
  • 하루 캐릭터: 진+모미+마음 통합 만능 비서, 모든 주제 대화 가능
  • Daily Insight: DB에서 건강/감정/스케줄 실시간 조합 (30분 갱신)
  • Recommendation: 하루 대화 내용 기반 팁 추출 + 체크 기능 (1시간 갱신)
  • 기록 Today: 날짜 스크롤 피커 + Health/Emotion/Schedule + Memo + Daily Review 4질문
  • 기록 Week: 달성률 카드 + 수면 바 차트 + 감정 흐름 + 한줄 요약
  • 기록 Month: 히트맵 캘린더 + 30일 차트(수면/컨디션/달성률) + 감정 분포 + 월간 요약
  • 컬러 룰: 블루(#60A5FA) = 달성/7h+, 그레이(#E6E6E6) = 미달성
  • Daily Review 4질문: 감사/완료/내일/나에게 한마디
친구들 → 하루 모미+마음 → 통합 비서 Week 대시보드 Month 대시보드 Recommendation 체크 Daily Review 4질문 달성률 DB 저장 컬러 통일 (블루+그레이) 모미/마음 독립 탭 친구들 탭

Stage 7 — 5/3~4 | design2 확장 (현재)
루틴 시스템 + 리포트/회고 + MI 공감 대화
습관 추적, 자동 리포트, 회고 시스템, 하루 대화 스타일 전면 개편. "기록하고 돌아보고 개선하는" 완전한 생산성 사이클 완성.
  • 루틴 시스템
    • 홈: Today 아래 Routine 섹션 — 체크 토글, 연속 일수, 🔥(3일+), "잊지 않으셨죠?"
    • 하루 대화로 루틴 추가/삭제 ("명상 루틴 추가해줘")
    • 기록 Today: 루틴 완료 상태 + 연속 기록 요약
    • 기록 Month: 루틴별 달성률 프로그래스 바
  • 주간/월간 리포트 (자동 문장)
    • Week: "이번 주는 7일 중 5일을 기록하셨어요. 수면은 평균 5.7시간으로..."
    • Month: "5월은 30일 중 20일을 기록하셨어요. 건강 면에서 평균 수면 6.2시간..."
    • 데이터 기반 자동 생성 — 수면, 컨디션, 운동, 감정, 루틴, Recommendation 달성률 종합
  • 회고 시스템 (직접 입력)
    • Week: 다음 주 다짐 + 한줄 소감
    • Month: 기억에 남는 순간 + 다음 달 목표 + 나에게 한 마디
    • DB 저장 (weekly_reviews, monthly_reviews)
  • 하루 MI 공감 대화
    • 동기면담(MI) 4원칙: 경청/반영, 열린 질문, 공감 먼저, 자율성 존중
    • 바로 리스트 금지 → "정리해드릴까요?" → 동의 후 제공
    • 체크박스: 실행 가능한 팁(3개+)일 때만, 체크된 것만 Rec/Todo 추가
    • 마크다운 렌더링 (**볼드**, *이탤릭*, `코드`)
    • 모바일 Enter = 줄바꿈 (전송은 버튼만)
    • 헤더 상단 고정 (fixed)
  • Recommendation 개선
    • 상단: 맥락 요약 ("수면이 부족해요, 수업이 있어요. 이런 부분을 챙겨보세요:")
    • 하단: 체크 가능한 실행 항목 + 달성 카운트
    • 하루 대화 내용 자동 추출 + 홈 탭 복귀 시 즉시 갱신
  • 기록 Today 개선
    • 스케줄: 오늘만 표시 (내일/모레 제거 — 홈에서 확인)
    • 하루 대화 요약: 주제별 인과관계 서술 ("출판에 대해 이야기함 — 접근성이 핵심")
    • 상투어 필터링 ("응", "해줘" 등 제외)
  • 기록 Week 확장
    • 달성률 카드: Recommendation + Todo 나란히
    • 일별 상세: 요일별 수면/컨디션/운동/감정
    • 운동 로그 + Recommendation 세부 항목
  • 기록 Month 확장
    • 캘린더: 요일 헤더(일~토), 월 네비게이션(←→), 폰트 확대
    • 차트: 마우스오버/터치 시 데이터 값 표시
    • 월간 요약: 건강/감정/루틴/활동 4섹션
  • Todo 시간 기록: completed_at 타임스탬프 저장
루틴 시스템 연속 일수 + 🔥 주간 리포트 (자동) 월간 리포트 (자동) 주간 회고 (입력) 월간 회고 (입력) 하루 루틴 관리 Todo 완료 시간 Rec 맥락 요약 마크다운 렌더링 하루 대화 → MI 공감 대화 요약 → 인과관계 서술 기록 Today 스케줄 → 오늘만 Week → 리포트+회고+상세 Month → 리포트+회고+루틴 딱딱한 리스트 응답
