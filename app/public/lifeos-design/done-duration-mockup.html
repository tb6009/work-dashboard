C · 결정해주실 항목

  1. 당일 표기 — 그대로 "당일" 표시 vs 생략 (대부분 당일이면 노이즈) ?
  2. 임계값 — 노란(주의) 7일+, 빨강(회고) 14일+ 가 적절한지 (3일/7일도 가능)
  3. 통계 카드 슬롯 — "평균 완료/일" 자리에 "평균 소요일"을 대체할지, 4열로 늘릴지
  4. 소팅 위치 — 토글을 통계 아래 칩으로 (시안) vs 네비 옆 작은 아이콘
  5. 회고 자동 picking — 14일+ 항목을 주간 리포트에 자동 인용 (지금은 별도 작업 — 향후 v0.10에서)
