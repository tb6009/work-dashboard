하루 기능

기능작동 방식
건강 기록"7시간 잤어" → 자동 DB 저장
감정 기록"막막해" → 자동 DB 저장
루틴 관리"명상 루틴 추가해줘" → routines 테이블
API 확인"API 확인" → 연결 상태 표시
도움말"뭐 할 수 있어?" → 기능 안내
Rec/Todo체크박스 → 선택 항목만 저장

MI 공감 대화 원칙


❌ 바로 리스트 → ✅ 먼저 공감 대화
❌ "다음 포인트를 고려하세요" → ✅ "어떤 부분이 고민이세요?"
❌ 한번에 5개 나열 → ✅ "정리해드릴까요?" 후 동의 시 제공

입력system (16px)iOS 확대 방지

인터랙션 원칙

  • 체크: 원형 토글 (빈 원 → 블루 채움 + 체크마크)
  • 저장: 버튼 클릭 → "저장 완료" (블루) 2초 표시
  • 날짜: scroll-snap 피커, 가운데 날짜=검은 배경+흰 텍스트
  • 차트: 마우스오버/터치 시 데이터 값 표시
  • 모바일 Enter: 줄바꿈 (전송은 파란 버튼만)

7. 데이터 아키텍처

Supabase 테이블 (12개)


health_logs        → 수면, 운동, 컨디션
journal_entries    → 감정, 메모, Daily Review 4질문, 감사
chat_messages      → 하루 대화 기록 (날짜별)
todos              → 할 일 (완료 시간 기록)
schedules          → 캘린더 일정
daily_insights     → 일일 인사이트
recommendation_logs → 추천 달성률
routines           → 루틴 항목 (추가/삭제)
routine_logs       → 루틴 일별 완료 기록
weekly_reviews     → 주간 회고
monthly_reviews    → 월간 회고
users              → 사용자 설정

데이터 흐름


사용자 → 하루 대화 → chat_messages + health_logs + journal_entries + routines
              ↓
         홈 탭 복귀 → Daily Insight (30분) + Recommendation (1시간) 갱신
              ↓
         기록 탭 → 날짜별 열람 + 주간/월간 리포트 자동 생성
              ↓
         회고 입력 → weekly_reviews / monthly_reviews 저장
