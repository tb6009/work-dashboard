개인 라이프 관리 앱

01
Design Principles
형태를 최소화하고 텍스트와 여백으로 구성합니다. 색은 시간의 흐름을 나타내는 데에만 사용합니다.
White Ground
배경은 순백. 콘텐츠가 스스로 위계를 만듭니다.
Text Only
100% Black으로 강조, 80% Gray로 본문. 장식 없이 글자만으로 구분합니다.
No Shape
카드, 테두리, 그림자 없이 여백과 구분선으로 구조를 잡습니다.
One Color
하나의 색이 시간에 따라 천천히 변합니다. 그 외에 색은 쓰지 않습니다.

03
Time Color
유일한 색. 시간대에 따라 하나의 색이 천천히 변합니다. 구분선, 액센트, 추천 블록에만 사용됩니다.
Option A / Blue Cycle
아침엔 옅은 하늘색, 밤으로 갈수록 깊어집니다. 가장 보편적이고 차분한 선택.
Option B / Warm Shift
아침 햇살에서 시작해 따뜻한 호박색으로. 자연광의 흐름을 따릅니다.
Option C / Ink
색 자체를 쓰지 않습니다. 먹의 농도처럼 시간에 따라 진해지기만 합니다.
하루 중 현재 시각의 색만 UI에 나타납니다. 한 화면에 여러 색이 동시에 보이지 않습니다.

Architecture
웹과 모바일이 동일한 DB와 테마 토큰을 공유합니다.
Frontend
Next.js (Web) + Expo (Mobile) / Turborepo monorepo
Shared
packages/types, packages/theme, packages/api / 타입, 테마 토큰, API 클라이언트 공유
Backend
Supabase / PostgreSQL + Auth + Realtime + Storage
Storage
Supabase Storage / 배경 이미지 업로드, 이미지 최적화
AI
Claude API / 캐릭터별 시스템 프롬프트, Edge Function 스트리밍
Theme
Design Tokens / 런타임 교체 가능, CSS Variables + RN StyleSheet

Database

// packages/theme/tokens.ts

export const text = {
  primary: "#000000",   // 100% black — 제목, 강조
  body: "#333333",      // 80% gray — 본문
  muted: "#808080",     // 50% — 보조
  faint: "#B3B3B3",     // 30% — 라벨
}

export const surface = {
  background: "#FFFFFF",
  divider: "#E6E6E6",
}

function getCurrentColor(palette, hour) {
  const index = Math.floor((hour / 24) * palette.length)
  return palette[Math.min(index, palette.length - 1)]
}
