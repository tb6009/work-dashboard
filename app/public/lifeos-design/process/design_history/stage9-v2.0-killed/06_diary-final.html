회고 작성 UI 형식 — 마지막 결정

주간회고·월간회고 모두 동일한 형식 적용

A

모달 (Bottom sheet, 추천)

캘린더 빨간 아이콘 탭 → 화면 하단에서 슬라이드업 모달로 회고 작성. 컨텍스트(달력) 유지. 모바일 친화. 저장하면 ✓ 체크로 변경.

B

별도 페이지

탭 → /diary/review/2026-W19 새 라우트 이동. 풀 화면 작성. 글이 길어질 때 유리. 컨텍스트 끊김.

C

같은 탭 안 인라인 펼침

Week 탭의 기존 "주간 리뷰" 영역을 그대로 활용. Month에서 회고 아이콘 탭하면 Week 탭으로 이동 + 해당 주차 회고 영역 자동 포커스.
